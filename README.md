# Recipe Collector

## Description
A full-stack web application designed to help users organize and manage their favorite recipes. This project focuses on CRUD operations and data persistence using a SQL database.

## Features
- **Add Recipes:** Easily input new recipes with titles and ingredients.
- **View All:** A clean dashboard to see your entire collection.
- **Search:** Quickly find recipes by name.
- **Responsive UI:** Works on both desktop and mobile browsers.

## Technologies Used
- **Backend:** Python 3, Flask
- **Database:** SQLite
- **Frontend:** HTML5, CSS3 

## Installation
1. Clone the repository: `git clone [YOUR-LINK-HERE]`
2. Install dependencies: `pip install flask`
3. Run the app: `python app.py`
